# Edra

## How to compile:
- specify SDL3_DIR in the build cache of CMake if needed.
- run make b{platform}{d/r} for an automated build.
    - platforms:
        - w - Windows
        - u - Unix
        - v - PSVita
    - d / r -> debug / release.
- make r for an automated run.
